I'm excited to share this project with you! This was my first task during my internship, where I was asked to create a landing page for a product. I chose the Laneige Lip Sleeping Mask because it fit perfectly with the design I had in mind. To make the website visually appealing, I used various image-editing tools like Canva, Figma, and IbisPaintX. Additionally, I utilized Bootstrap CSS for responsive design, and incorporated a bit of JavaScript to style the page and make some elements interactive.
